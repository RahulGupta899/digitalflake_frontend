# Digitalflake_Admin_Frontend

## Clone the Repository
```bash
git clone https://github.com/RahulGupta899/digitalflake_frontend
```

## Change directory
```bash
cd digitalflake_frontend
```

## Install Packages
```bash
npm install
```

## Run the Application
```bash
npm run dev
```
